# 🍷 Adega Nat AI

Sistema inteligente para adegas com catálogo digital, carrinho interativo e pedidos automáticos via WhatsApp.

## 🚀 Sobre o Projeto

O Adega Nat AI foi criado para modernizar o atendimento de adegas e deliverys de bebidas, oferecendo uma experiência digital prática, rápida e intuitiva.

O sistema permite que clientes visualizem produtos, adicionem itens ao carrinho e finalizem pedidos diretamente pelo WhatsApp.

Além disso, o projeto está sendo estruturado para incluir:

- Controle inteligente de estoque
- Painel administrativo
- Automação de atendimento
- Recomendações com IA
- Dashboard de vendas
